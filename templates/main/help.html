Yardım sayfası

Sniffer

Yerel ağınızdaki paketleri dinleyebileceğiniz web arayüzü sayfası. Paketlerin filtrelenmesi için BPF(Berkeley Packet Filter) syntax'ı kullanılmaktadır. Bilgi almak için BPF Syntax

Örneğin aşağıdaki filtre ifadesi, IP adres kaynağı 192.168.1.35 olan paketleri ifade eder.
ip src 192.168.1.35

Paketlerin detayını görmek için, ilgili pakete çift tıklamanız yeterlidir. Sağ taraftaki panelde paketin detayları gösterilecektir.

Analyzer

Sniffer tarafından yakalanan paketlerin sayısal bilgi olarak chart grafik üzerinde gösterildiği web arayüzü. Aynı zamanda iki adres arasındaki birebir iletişimin detayları da görülebilir.
